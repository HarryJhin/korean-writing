# im-not-ai taxonomy 갭 심사 (2026-07-03)

> **목적**: im-not-ai(humanize-korean) taxonomy 60+ 패턴 중 korean-writing 규칙(R1~R6)이
> 커버하지 않던 갭을 심사해, 검증 기준을 통과한 것만 규칙으로 수입한다.
> **방법**: 발견 패스(전 패턴 나열 + 출처 등급) ↔ 필터 패스(수입 판정) 분리.
> **수입 기준**: (1) 근거 등급, (2) 생성 시 지침 적합성(사후 탐지 전용 신호 배제),
> (3) 오탐 위험, (4) 기존 R1~R6과의 중복 여부.

## 검증 노트 (1차 확인)

- **연결어미 뒤 쉼표 4.84배**: KatFishNet Appendix F.1에서 verbatim 확인. 에세이 기준
  인간 4.10% vs LLM 19.83%. 시 15.57%/4.68%, 초록 28.01%/13.27%. **수입 가능(수치 포함)**.
- **쉼표 분절 평균 길이**: 논문 Table 2는 **형태소(morphemes)** 단위(에세이 인간 4.35 vs
  LLM 8.56). im-not-ai의 "어절" 표기는 전사 오류. 수입 시 단위 교정.
- **"KatFish 검증 결산 lexicon 4종(결론적으로·따라서·이를 통해·그러므로)"**: 논문에서
  **미확인**. 논문은 쉼표 위치 관련 접속부사(그리고·그러나·그런데·그러므로)만 언급하고
  빈도 lexicon을 제공하지 않는다. 검증 딱지 없이 관찰 기반 어휘군으로 격하 수입.
- **대명사 밀도 2~3배(김도훈 2009)**: im-not-ai 내부 보고서 verbatim만 존재, 원논문
  1차 미확인. 패턴은 수입하되 수치는 미인용.

## 수입 결과 요약

- **S2 마커 신설 4종** (결정론 카운트, 훅 게이트 아님): 연결어미 뒤 쉼표, 가지고 있다,
  이중 조사, 무맥락 결산 상투구
- **R2 확장**: 번역투 마커 표 (있어서/관련하여류, have 직역, 자동 피동, 이중 조사,
  대명사 직역, 명사화 접미사, 연결어미 뒤 쉼표)
- **R3 보강**: 단문 일변도, 쉼표 긴 절 사슬
- **R4 확장**: 슬롭 어휘군 표 9종 + 형식명사·문두 접속사·hedging·수식 중복·종결어미 항목
- **R7 신설**: 서식·강조 절제 (볼드·따옴표·괄호·병기·인덱싱·콜론 헤딩·불릿·안내문)
- **R1·R5·R6 골격 유지** (S1 4종 불변, 훅 동작 불변)

수치는 1차 확인분만 인용한다(KatFish 4.84배). 원본의 회차 관찰 수치·모델 우세
딱지·"KatFish 검증 lexicon" 딱지는 수입하지 않는다.
